Wanderlust

Wanderlust

Wanderlust is a mini travel app that uses location specific searches to suggest venues for travellers to visit.

The tool that were used to create this project:

HTML

Sass

CSS3

SVG

jQuery

JavaScript

AJAX

JSON

Web APIs


This is web app that employs the use of APIs from data providers such as Foursquare to create a mini travel guide with provides local venue and weather information of almost any city in the world (it'll only show 4 results at a time as I only get 50 free per 24 hours).

I created the UI and logo from scratch using Sass and an SVG editor (Inkscape). I also used a type-ahead JavaScript library created by twitter which I integrated with an external database from datahub.io in the form of a JSON file to retrieve global cities. This project can be viewed on a desktop screen and mobile.

Jammming

Jammming

Jammming is a web app that utilised the Spotify API to allow users to log into their accounts and create custom playlists that can be saved.

The tool that were used to create this project:

HTML

CSS3

React

Hooks

NPM

AJAX

Web APIs


This is one of my slightly more complex projects using React components, passing state, and making API requests to create the web app.

Using the Spotify API, a user can search artists, songs, and albums to save their custom playlists on to personal user accounts (Spotify login required). It has a responsive layout and the project is deployed using surge.sh, a static web publishing tool.

Magic EightBall

CSS3

SVG


The concept is simple but I spent some time planning and designing the layout, which was written in Sass. An SVG was used for the eightball, customised a combinations of fonts and decorative effects, and used timed transitions to create animations.
